# 📲 Telegram Reminder Automation (n8n)

This project demonstrates a simple automation workflow that sends reminder messages using Telegram Bot API.

## 🚀 Features
- Automated message sending
- Delay-based scheduling
- API integration using HTTP Request
- Built using n8n workflow automation

## ⚙️ Workflow
Trigger → Wait → Send Message (Telegram API)

## 🛠️ Tech Stack
- n8n
- Telegram Bot API
- HTTP Requests

## 📸 Output
The system sends a reminder message automatically after a defined delay.

## 🔮 Future Improvements
- Integration with WhatsApp API
- Use of real-time data (Google Sheets / Database)
- Scheduled triggers instead of manual execution
